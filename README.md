# GPS Map Camera Android App
Developed a GPS map camera app for Android, integrating camera functionalities with location-based services. Enabled real-time GPS tagging for photos, overlaying location details and google map with current location marker and the option of changing the map types into four types of maps on captured images. The app also has the feature of saving the captured image into three different directories of the device storage. Included features for switching between front and back cameras, applying filters, and adjusting brightness. Implemented a settings screen for customizing camera sounds . Conducted extensive testing to ensure performance across various devices.
